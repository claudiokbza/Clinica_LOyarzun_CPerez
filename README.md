# Clinica_LOyarzun_CPerez
Proyecto de EVA 2 Frontend
## Descripción

Este proyecto es una aplicación frontend desarrollada como parte de la evaluación EVA 2 para el curso de Front End. Permite gestionar información relacionada con la clínica, como pacientes, citas y profesionales de la salud.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Instalación

1. Clona este repositorio:
    ```bash
    git clone https://github.com/usuario/Clinica_LOyarzun_CPerez.git
    ```
2. Ingresa al directorio del proyecto:
    ```bash
    cd Clinica_LOyarzun_CPerez
    ```
3. Abre el archivo `index.html` en tu navegador.

## Uso

- Navega por la aplicación para gestionar pacientes, citas y profesionales.
- Utiliza las funcionalidades disponibles según los requerimientos del proyecto.
